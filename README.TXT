PROPOSITO DO PROJETO:
Trabalho realizado para a UC de Programação Orientada a objetos do curso Engenharia Informática do Instituto politécnico de Beja.
Este programa tem o intuito de escolher um dos ficheiros ".txt" que se encontram na pasta resources já tratados num formato especifico,
de modo a que mostre algum tipo de métrica ao longo dos anos ou meses.
Por exemplo, no ficheiro "brands.txt", mostra um corrida de barras ao longo do tempo,
mostrando a evolução dos valores das marcas ao longo dos meses mostrando sempre o top 12 das marcas.

COMO FUNCIONA:
Ao iniciar o main do BarCharRacerStart, abrirá uma janela para escolher um dos ficheiros da pasta resource apos isso é redircionado
para um ecrã com um menu com dropdown no canto superior esquerdo, onde é possivel iniciar a "corrida", sair do ecrã ou gerar um ficheiro
com algumas estatisticas chamado "StatsData.txt" encontrado na pasta "generatedData".
